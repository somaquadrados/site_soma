Introduo

Esse mini-tutorial visa ensinar a como plotar box plot e o violin plot no mesmo grfico usando o ggplot2. Nas sesses 1 e 2 mostramos o que um box plot e um violin plot e discutimos um pouco sobre suas vantagens. Na sesso trs, mostramos como plotar os dois juntos.

Box plot

Boxplot, diagrama de caixa, ou diagrama de quartis, ou meu favorito, grfico de caixa de bigodes, um grfico que permite visualizar como o seu conjunto de dados est distribudo. um grfico clssico em que mediana o centro da caixa e o primeiro quartil e o terceiro formam o retngulo. Por ltimo, bigodes da caixa so formados pelo limite inferior e o limite superior. Tipicamente, os outliers so representados por pequenos pontos no grfico. Do wikipedia:

Por CEPID NeuroMat, CC BY-SA 4.0, https://commons.wikimedia.org/w/index.php?curid=56047497

Violin plot

Violin plot uma visualizao similar a um box plot, porm ela tem a vantagem de usar um kernel de densidade simtrico em cada lado. Ou seja, ela tambm consegue refletir a densidade de probabilidade (ou em outras palavras, aquela linha que costuma estar cima dos histogramas). A maior vantagem que ela consegue ser mais informativa que um simples diagrama de caixa porque ela mostra o seu conjunto inteiro de dados. Infelizmente, no um grfico to popular o que pode gerar um pouco de confuso sobre sua interpretao. Tambm do wikipedia:

Por itl.nist.gov - VIOLIN PLOT at www.itl.nist.gov, Public Domain, https://commons.wikimedia.org/w/index.php?curid=4992563

Por que usar ambos juntos?

O pulo do gato que voc plotar ambos no mesmo grfico. Esse reflexo especialmente importante se seus dados so multimodais porque isso fica evidente usando o violin plot e no claro no box plot. Outra vantagem que isso evita tambm que voc acabe com um box plot, um histograma + estimativa de densidade kernel para cada varivel. Pela prpria natureza dos grficos, tambm mais fcil de visualizar a sua amostra por fatores diferentes.

PlantGrowth

Usando ggplot2, voc pode plotar dois grficos diretamente usando apenas somando dois objetos geomtricos diferentes, mas o resultado no muito simptico.

ggplot(PlantGrowth, aes(x=weight, y=group, color = group)) + 
        geom_violin() +
        geom_boxplot()

O primeiro passo mudar a largura dos boxplots. Para isso, passamos o width = 0.3 como parmetro dentro da chamada do boxplot. Tambm mudados o tema usando theme_light para o grfico ficar mais coeso.

ggplot(PlantGrowth, aes(x=weight, y=group, color = group)) + 
  theme_light() +
  geom_violin() +
  geom_boxplot(width=0.3)

Por ltimo, voc pode escolher colocar o grfico de p. Nesse caso, basta pedir para que as coodernadas sejam invertidas usando coord_flip().

ggplot(PlantGrowth, aes(x=weight, y=group, color = group)) + 
  theme_light() +
  geom_violin() +
  geom_boxplot(width=0.3) +
  coord_flip()

Insect Sprays

Podemos fazer a mesma coisa usando o dataset InsectSprays.

ggplot(InsectSprays, aes(x=spray, y=count, color = spray)) + 
  theme_light() +
  geom_violin() +
  geom_boxplot(width=0.05)
